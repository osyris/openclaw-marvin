# Idea #001 — Upcycled Bottle Glasses

**Добавлена:** 2026-03-16  
**Статус:** Идея / на изучении

## Суть
Бутылки (пиво, вино) нарезаются газовой резкой / специальным инструментом и полируются → превращаются в дизайнерские стаканы. Нижняя часть = стакан, верхняя = блюдце/подставка.

На фото: зелёная бутылка Heineken → стакан + подставка. Выглядит как готовый продукт.

## Модель KZ
- **KZ не делает сам** — передаёт производство исполнителям (мастер, мастерская)
- **KZ = дистрибуция + продажи**: маркетплейсы (Etsy, местные), рестораны/кафе, gift shops, B2B (корпоративные подарки)

## Потенциальные каналы сбыта
- Etsy / Shopify (eco-friendly/upcycled ниша горячая)
- Локальные рестораны, кафе, бары — брендированные стаканы из их же бутылок
- Корпоративные подарки (бутылка → стакан с логотипом партнёра)
- Эко-ярмарки, маркеты

## Что нужно проверить
- [ ] Найти мастера/мастерскую (Таллин или EU) с нужным оборудованием
- [ ] Себестоимость 1 стакана (материал + работа + полировка)
- [ ] Ценовой диапазон на Etsy (похожий товар)
- [ ] Объём рынка / конкуренция
- [ ] Логистика (хрупкий товар = дорогая упаковка)

## Плюсы
- Низкая себестоимость сырья (бутылки бесплатны или копейки)
- Eco/upcycled тренд устойчив
- Узнаваемость бренда бутылки = маркетинговый крючок
- Можно делать кастомные под заказ (Heineken, Corona и т.д.)

## Риски
- Конкуренция на Etsy уже есть
- Хрупкость = возвраты, упаковка
- Масштабирование ограничено ручным трудом
